### 프록시 (devetym-proxy 코드 상수 — feat/cost-hardening, ⏳ 배포 대기)

| 항목 | 값 | 사용자에게 보이는 것 |
|---|---|---|
| AI 검색 횟수 (기기당) | **10회/일** (`DAILY_LIMIT=10`, UTC 자정=KST 09:00 리셋) | 429 → "오늘 사용량을 모두 사용했어요" |
| AI 검색 횟수 (전역, 전 기기 합산) | **200회/일** (`GLOBAL_DAILY_LIMIT=200`) | 동일 429 문구 |
| 서비스측 소진 (크레딧 0·월 한도 도달) | Anthropic 결제 에러 → **402** 매핑 | "AI 생성에 문제가 있어요. 잠시 후 다시 시도해주세요" |
| 과금 파라미터 서버 강제 | model=claude-sonnet-4-6 · max_tokens≤4096 · thinking budget≤2000 · 본문≤32KB | (해당 없음 — 위조 방어) |

### 방어 층 요약 (돈이 새는 순서대로 브레이크)

1. 기기당 10회/일 (프록시 KV)
2. 전역 200회/일 (프록시 KV) — 하루 최대 대략 $8 수준에서 차단
3. 이메일 알림 $10 → $20 → $25 (사람 대응 시간)
4. **월 지출 한도 $30 (Console — 최종 하드캡)** → 도달 시 사용자에겐 402 문구, 매월 1일 자동 해제
5. 크레딧 자동 리로드($5→$15)는 상한이 아니라 **유동성** — 서비스가 예측 불가능한 시점에 끊기는 것을 방지
