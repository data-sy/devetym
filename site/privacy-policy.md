# 개인정보 처리방침 — 개발 어원 사전 (DevEtym)

본 방침은 "개발 어원 사전"(이하 "앱")이 수집·처리하는 개인정보의 범위와 목적을 설명합니다. 본 앱은 **Android·iOS 양 플랫폼**으로 제공됩니다. 최종 갱신일: 2026-07-05.

## 1. 수집 항목

앱은 **사용자가 명시적으로 동의한 경우에 한해** 다음 익명 이용 데이터를 수집합니다.

| 항목 | 설명 | 예시 |
|---|---|---|
| 검색 키워드 | 사용자가 입력한 영문·한글 용어 | `mutex`, `뮤텍스` |
| 검색 시각 | 이벤트 발생 시각 (UTC) | `2026-07-05T14:32:10Z` |
| 검색 결과 종류 | 번들 DB 히트 / AI 생성 / 미검색 용어 / 오타 추정 | `bundle_hit`, `ai_generated`, `not_dev_term`, `possible_typo` |
| 오류 종류 | API 오류 발생 시 분류 코드 | `timeout`, `invalid_response`, `network_error` |
| 익명 디바이스 식별자 | Firebase가 생성하는 App Instance ID (앱 재설치 시 변경, 사용자 실명·이메일 등과 연결 불가) | `abc123...` |

**수집하지 않는 것:**
- 이름, 이메일, 전화번호 등 직접 식별 정보
- 디바이스의 광고 식별자(iOS IDFA·Android AAID) 또는 OS 레벨 기기 ID
- 위치 정보
- 다른 앱이나 웹 활동 기록
- 검색 결과로 열람한 상세 내용, 북마크 목록, 검색 히스토리 (이들은 모두 기기 내에만 저장)

## 2. 수집 목적

수집된 데이터는 오직 다음 목적으로만 사용됩니다.

- 어떤 개발 용어가 자주 검색되는지 파악하여 번들 사전 확장 우선순위에 반영
- AI 생성 응답의 오류·실패율을 관찰하여 프롬프트·모델을 개선
- 앱 전체 이용량(주간 활성 사용자 수) 집계

**수집된 데이터를 제3자에게 판매하거나 광고 목적으로 사용하지 않습니다.**

## 3. 보관 기간

- 수집 이벤트: **수집일로부터 14개월** 이후 자동 삭제 (Firebase Analytics 기본 보관 기간)
- 사용자가 앱 내에서 "데이터 수집 동의"를 철회하면 이후 이벤트는 더 이상 수집되지 않습니다. 이미 수집된 과거 이벤트의 삭제는 4번 절차를 참고하세요.

## 4. 사용자 권리

동의 철회 및 데이터 삭제를 언제든 행사할 수 있습니다.

- **동의 철회**: 앱의 [설정 → 데이터 수집] 화면에서 토글을 OFF로 변경. 이후 이벤트 수집이 즉시 중단됩니다.
- **과거 데이터 삭제 요청**: 아래 연락처로 이메일 요청 시, 해당 디바이스의 App Instance ID에 연결된 모든 이벤트를 삭제합니다. 요청 시 이메일에 해당 ID를 포함해주세요 (ID는 [설정 → 데이터 수집 → 내 식별자 보기]에서 확인 가능).
- **앱 삭제**: 앱을 삭제하면 App Instance ID는 무효화되며, 이후 해당 ID로 수집된 데이터는 개인과 연결할 수 없게 됩니다.

## 5. 수탁·국외 이전

앱은 Google의 Firebase Analytics를 데이터 처리 수탁자로 이용합니다.

- 수탁자: Google LLC
- 이전 항목: 1번에 명시된 모든 수집 항목
- 이전 국가: 미국 (Firebase Analytics 서버)
- 이전 일시 및 방법: 이벤트 발생 시점에 HTTPS 암호화 전송
- 보유·이용 목적 및 기간: 본 방침 2·3번과 동일

Google의 자체 개인정보 처리방침은 https://policies.google.com/privacy 을 참고하세요.

## 6. 아동 개인정보

앱은 만 14세 미만 아동을 대상으로 하지 않습니다. 만 14세 미만 사용자의 개인정보를 의도적으로 수집하지 않으며, 수집 사실이 확인되는 경우 즉시 삭제합니다.
