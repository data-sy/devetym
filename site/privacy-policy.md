# 개인정보 처리방침 — 개발 어원 사전 (DevEtym)

본 방침은 "개발 어원 사전"(이하 "앱")이 처리하는 데이터의 범위와 목적을 설명합니다. 본 앱은 **Android·iOS 양 플랫폼**으로 제공됩니다. 최종 갱신일: 2026-07-06.

> **요약:** 앱은 **개인을 식별하는 정보를 수집하지 않습니다.** 북마크·검색 히스토리·열람 내용은 **모두 기기 내에만** 저장됩니다. 단, 번들 사전에 없는 용어를 검색할 때에 한해, 해당 기능(AI 생성)을 제공하기 위해 **검색어**가 서버(프록시)로 전송됩니다. 현재 버전은 사용 통계(애널리틱스)를 수집하지 않습니다.

## 1. 기기 내에만 저장되는 항목 (전송하지 않음)

다음 항목은 **사용자 기기 안에만** 저장되며, 외부로 전송하거나 제3자와 공유하지 않습니다.

- 북마크 목록
- 검색 히스토리(과거 검색어·검색 시각)
- 검색 결과로 열람한 용어 상세 내용
- 외관(라이트/다크/시스템) 등 앱 설정

앱을 삭제하면 위 데이터는 기기에서 함께 삭제됩니다.

## 2. 앱 기능 제공을 위해 전송되는 항목 (AI 생성 검색)

앱은 650개 이상의 용어를 **오프라인 번들 사전**으로 내장하고 있어, 대부분의 검색은 네트워크 없이 기기 안에서 처리됩니다. 검색한 용어가 **번들 사전에 없을 때에 한해**, 정의를 생성하기 위해 다음이 서버로 전송됩니다.

| 항목 | 설명 | 목적 |
|---|---|---|
| 검색 키워드 | 번들에 없어 AI 생성이 필요한 용어 | 해당 용어의 정의·어원 생성 |
| 기기 식별자 (`X-Device-Id`) | 앱이 기기별로 생성하는 임의 식별자 (앱 재설치 시 리셋, 사용자 실명·이메일 등과 연결 불가) | 남용 방지를 위한 **일일 요청 한도** 계수 |

- 전송 방식: **HTTPS 암호화**로 앱 전용 프록시 서버(Cloudflare Worker)를 경유해 Claude API에 도달합니다.
- 위 항목은 앱 기능 제공(정의 생성·한도 관리) 목적으로만 쓰이며, **사용 통계나 프로파일링 목적으로 저장·분석하지 않고, 제3자에게 판매하거나 광고에 사용하지 않습니다.**
- 기기 식별자는 앱 안에서만 생성되는 임의 값으로, OS 광고 식별자나 실명·계정과 연결되지 않습니다.

## 3. 수집하지 않는 것

앱은 다음을 **수집하지 않습니다.**

- 이름, 이메일, 전화번호 등 직접 식별 정보
- 광고 식별자(iOS IDFA · Android AAID) 또는 OS 레벨 기기 ID
- 위치 정보
- 다른 앱이나 웹 활동 기록
- **사용 통계(애널리틱스).** 현재 버전은 Firebase Analytics 등 어떤 분석 도구도 통합하지 않으며, 검색 통계·이용량 데이터를 수집하지 않습니다.

## 4. 향후 애널리틱스 도입 시 (현재 미적용)

앱에는 사용 통계 수집을 위한 **동의 화면(설정 → 데이터 수집)**이 마련되어 있으나, **현재 버전에서는 실제로 수집되는 데이터가 없습니다.** 향후 업데이트에서 익명 사용 통계(예: 어떤 용어가 자주 검색되는지) 수집을 도입할 경우:

- 본 방침을 먼저 갱신하고, 앱 내에서 **명시적 동의를 받은 경우에만** 수집합니다.
- 그 시점의 수집 항목·목적·보관 기간·수탁자를 본 방침에 구체적으로 명시합니다.

동의하지 않아도 앱의 모든 핵심 기능(검색·AI 생성·북마크·히스토리)은 정상 이용할 수 있습니다.
